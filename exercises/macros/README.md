# 宏

Rust 的宏系统非常强大，但也有点难以理解。
我们不会教你如何写出自己的特色宏。
相反，我们将会向你展示如何使用和创建它们。

如果你想要学习更多关于编写自己的宏的内容，
[macrokata](https://github.com/tfpk/macrokata) 项目有类似 Rustlings 风格的练习，
不过它全是关于如何写宏的。
