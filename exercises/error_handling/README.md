# 错误处理
大多数的错误都不严重到需要程序完全退出。
有时，当一个功能失败时，其原因很容易解释和相应。
比如，当你试图打开一个文件，但是操作因为文件不存在而失败了，你可能想要创建文件而不是终止程序。
