# 泛型

泛型是将类型和功能推广到更广泛情况的说法。
这对于在许多方面减少代码重复非常有用，但是可能需要相关语法。
也就是说，泛型需要小心地指定哪些类型是这个泛型真正认为有效的类型。
最简单也最常见的泛型使用方法是作为类型参数。
